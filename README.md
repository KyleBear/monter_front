# Monter Front

바닐라 자바스크립트로 구성된 프론트엔드 프로젝트입니다.

## 파일 구조

- `index.html`: 메인 HTML 파일
- `style.css`: 스타일시트
- `script.js`: 자바스크립트 로직

## 실행 방법

이 프로젝트는 정적 파일로 구성되어 있습니다. 다음 방법 중 하나로 실행할 수 있습니다:

1. `index.html` 파일을 웹 브라우저로 직접 엽니다.
2. VS Code의 `Live Server` 확장을 사용합니다.
3. Python이 설치되어 있다면: `python -m http.server`
4. Node.js/npm이 설치되어 있다면: `npx serve .`
